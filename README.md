# SIV — Steam Inventory Viewer

[![Release](https://github.com/MeinLiX/SIV/actions/workflows/release.yml/badge.svg)](https://github.com/MeinLiX/SIV/actions/workflows/release.yml)
![Latest Release](https://img.shields.io/github/v/release/MeinLiX/SIV?label=version)

A desktop app for viewing and analyzing your Steam inventory. Currently supports Counter-Strike 2, with more games potentially coming in the future.

## Features

- **Inventory browsing** — view all items with grouping, search, sorting & filtering
- **Item details** — float, paint seed, stickers, keychains, rarity, wear condition, origin
- **Price monitoring** — fetch live prices from Steam Market with local caching
- **Storage Units** — browse casket contents and estimate their value
- **Steam authentication** — login/password, QR code, Steam Guard, session persistence
- **Multi-account** — save and switch between multiple accounts
- **Auto-update** — check and install new versions from GitHub
- **Themes** — Dark / Light / System

## Download

> **[Download latest release](https://github.com/MeinLiX/SIV/releases/latest)**

Two build variants are available:

| Variant | Size | Requirements |
|---------|------|--------------|
| **Self-Contained** | ~larger | Everything included, no .NET installation needed |
| **Framework-Dependent** | ~smaller | Requires [.NET 10 Desktop Runtime](https://dotnet.microsoft.com/download/dotnet/10.0) |

If unsure — pick **Self-Contained**.
